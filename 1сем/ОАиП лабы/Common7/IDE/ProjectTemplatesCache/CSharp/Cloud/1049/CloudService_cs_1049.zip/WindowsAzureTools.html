Пакет Microsoft Azure SDK для .NET

Чтобы включить поддержку для построения масштабируемых веб-приложений и служб в Microsoft Azure, установите пакет Microsoft Azure SDK для .NET.

Пакет Microsoft Azure SDK для .NET расширяет Visual Studio, добавляя возможность создания, построения, отладки, выполнения и упаковки масштабируемых веб-приложений и служб в Microsoft Azure.

Примечание: проект, содержащий эту страницу, предоставляет только инструкции по установке требуемого набора инструментов.
