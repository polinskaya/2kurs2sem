Установка Windows 8.1

Чтобы разрабатывать приложения для Магазина Windows, выполните обновление до Windows 8.1.

Для разработки приложений для Магазина Windows в Visual Studio 2013 необходима операционная система Windows 8.1.

Примечание. Проект, содержащий эту страницу, предоставляет только инструкции по установке Windows 8.1.
