Чтобы использовать этот пользовательский элемент управления в файле XAML в другом проекте, выполните следующие действия:

  • Добавьте ссылку на этот проект.
  • Добавьте в корневой элемент файла XAML, в котором требуется использовать этот элемент управления, следующую строку:
    xmlns:MyNamespace="clr-namespace:$safeprojectname$;assembly=$safeprojectname$"
  • Используйте элемент управления в файле XAML:
    <MyNamespace:CustomControl1/>
