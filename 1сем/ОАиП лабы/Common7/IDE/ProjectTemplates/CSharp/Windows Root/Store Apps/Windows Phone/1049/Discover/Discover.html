Пакет Windows Phone SDK 8.0

Приступайте к сборке приложений для Windows Phone 8 или Windows Phone 7.5 с помощью пакета Windows Phone SDK 8.0.

Пакет Windows Phone SDK 8.0 расширяет возможности Visual Studio 2012, предоставляя необходимые средства для разработки приложений для Windows Phone, включая шаблоны приложений и библиотек Windows Phone, эмулятор Windows Phone и средства анализа производительности. С помощью этого расширения Visual Studio можно компилировать и отлаживать приложения Windows Phone, а затем развертывать их в Windows Phone или эмуляторе Windows Phone.

Примечание    Проект, в котором отображается эта страница, содержит инструкции по установке пакета Windows Phone SDK 8.0, но не сам пакет SDK. Чтобы скачать пакет SDK, нажмите кнопку Скачать пакет Windows Phone SDK 8.0 или перейдите по адресу [URL страницы скачивания].
